# ============================================================
# CodeSearch AI — System Requirements & Setup Guide
# ============================================================
# This is a Next.js (Node.js) project, not Python.
# Follow the steps below to run it on any PC.
# ============================================================


# ------------------------------------------------------------
# STEP 1 — Install Node.js (v18 or higher recommended)
# ------------------------------------------------------------
# Download from: https://nodejs.org/en/download
# Verified working on: v24.13.0
#
# After install, verify:
#   node --version
#   npm --version


# ------------------------------------------------------------
# STEP 2 — Install pnpm (package manager used by this project)
# ------------------------------------------------------------
# Run in terminal:
#   npm install -g pnpm
#
# After install, verify:
#   pnpm --version


# ------------------------------------------------------------
# STEP 3 — Install project dependencies
# ------------------------------------------------------------
# Navigate to the portfolio folder and run:
#   cd portfolio
#   pnpm install


# ------------------------------------------------------------
# STEP 4 — Set up environment variables
# ------------------------------------------------------------
# Create a file named .env.local inside the portfolio/ folder:
#   portfolio/.env.local
#
# Add the following line with your actual Groq API key:
#   GROQ_API_KEY=your_groq_api_key_here
#
# Get a free Groq API key at: https://console.groq.com


# ------------------------------------------------------------
# STEP 5 — Run the development server
# ------------------------------------------------------------
#   cd portfolio
#   pnpm dev
#
# Then open in browser: http://localhost:3000
